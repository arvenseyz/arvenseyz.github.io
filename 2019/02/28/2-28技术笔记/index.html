注解

ApplicationContext

具体怎么做呢？基本思想就是，为每一个切面所切入的类创建代理类，代理类来执行。那么区别就是，上下文里存放的不是被代理的类，而是操纵类。

代理

使用反射的方法，在被拦截方法的前后，执行切面的doAfter和doBefore方法。
