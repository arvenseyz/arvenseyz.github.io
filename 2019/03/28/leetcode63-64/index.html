题目

最小路径：

Input:
[
[1,3,1],
[1,5,1],
[4,2,1]
]
Output: 7
Explanation: Because the path 1→3→1→1→1 minimizes the sum.

解答

其实和之前那道路径数差不多，相当于带权的，然后比较即可。

到一个点的最短距离，等于【到他上面然后往下走一步】和【到他左边往右走一步】两者中较小的一个。

这就是动规方程咯。
