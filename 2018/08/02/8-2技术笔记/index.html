Preconditions

Preconditions类似与断言，使用其可以做参数校验

  1. checkNotNull(T reference)：校验对象是否为空。
  2. checkNotNull(T reference, @Nullable Object errorMessage)：对象为空时显示指定的错误信息。
  3. checkNotNull(T reference, @Nullable String errorMessageTemplate,@Nullable Object… errorMessageArgs)：允许在错误信息中使用占位符。

StringUtils

lamdba

表达式格式

lamdba表达格式为
(parameters) -> expression
或
(parameters) ->{ statements; }

λ表达式本质上是一个匿名方法。
原方法

转化为lamdba表达式
(int x,int y) -> { return x + y; }
省略参数类型和return
(x,y) -> x+y;
如果只有一个参数且可以被Java推断出类型，那么参数列表的括号也可以省略：
x -> x+1;
